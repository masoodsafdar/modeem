FAQ

Can i run this theme on Odoo v8?

No. If you are interested Material Backend Theme v8 - click here.

During installation I get an error: "You are trying to install incompatible themes:"

1. Please edit a file '__openerp__.py' (main directory) and change the line:.
'category' : 'Theme/Backend', with 'category' : 'Website',
2. Next, run 'Update Apps List' from developer mode.
3. Now you can install the theme.
